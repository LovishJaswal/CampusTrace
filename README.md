# CampusTrace

CampusTrace is a MERN-based lost and found portal designed for students to report, track, and manage lost items on campus.
Built with heavy AI assistance while learning MERN architecture just to understand the workflow

---

## Features

- User signup and login
- Authentication system
- Dashboard interface
- Report lost or found items
- Recent reports section
- MongoDB database integration
- Responsive modern UI

---

## Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose

## Current Status

The project is currently under development. Core authentication, dashboard, backend APIs, and MongoDB connectivity are working.

Future improvements may include:

## Purpose of This Project

The main goal of this project was to learn how frontend and backend systems connect in a real-world MERN application while understanding authentication, APIs, database integration, and application structure.
